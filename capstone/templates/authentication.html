Authentication API Documentation

/register

Method: POST

Description: Register a new user.

Request Body:

{ "username": "sampleUsername", "email": "sampleEmail@example.com", "password": "samplePassword" }

Response: User created successfully with a 201 status code.

/login

Method: POST

Description: Log in with a username and password.

Request Body:

{ "username": "sampleUsername", "password": "samplePassword" }

Response: Returns an access token if the username and password are valid.

{ "access_token": "generatedTokenHere" }

Usage commands:

Using curl to interact with your API:

Register:

curl -X POST http://localhost:5000/register \ -H "Content-Type: application/json" \ -d '{"username":"sampleUsername", "email":"sampleEmail@example.com", "password":"samplePassword"}'

Login:

curl -X POST http://localhost:5000/login \ -H "Content-Type: application/json" \ -d '{"username":"sampleUsername", "password":"samplePassword"}'
