Flask Weather & IP Information API


This API provides endpoints to retrieve weather information based on a city and IP-related details of the server.

1. Get Weather for a City

Endpoint: /getWeather/<city>

Method: GET

URL Parameters:

  • city (required): Name of the city for which you want to fetch the weather.

Response: A JSON response containing weather details for the given city.

Example Request:

GET /getWeather/London

Example Response:

{
   "coord": {...},
   "weather": [...],
   ...
}

2. Get IP Information

Endpoint: /api/ipinfo

Method: GET

Response: A JSON response containing details about the IP, hostname, city, region, country, location coordinates, organization, postal code, and timezone of the server.

Example Request:

GET /api/ipinfo

Example Response:

{
   "ip": "123.45.67.89",
   "hostname": "example.com",
   "city": "Sample City",
   ...
}

3. Get Server's Location (City)

Endpoint: /api/userlocation

Method: GET

Response: A JSON response containing the city in which the server is located.

Example Request:

GET /api/userlocation

Example Response:

{
   "city": "Sample City"
}

Or, in case the city is not found:

{
   "error": "City not found"
}
