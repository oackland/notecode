Weather API Documentation

This documentation provides an overview of the /api/weather endpoint, which retrieves weather data for the server's location.

Get Weather for Server's Location

Endpoint: /api/weather

Method: GET

Response:

A JSON response containing various weather-related details for the server's location.

Response Fields:

  • weather: General weather description (e.g., "clear sky").
  • pressure_in_bar: Atmospheric pressure in hPa.
  • temperature_celsius: Current temperature in Celsius.
  • temperature_fahrenheit: Current temperature in Fahrenheit.
  • icon: Weather icon code (you can use it to fetch icons from OpenWeather's platform).
  • wind: Wind speed in m/s.
  • sunset: Time of sunset (UNIX timestamp).
  • sunrise: Time of sunrise (UNIX timestamp).
  • temp_high_for_day: Highest temperature recorded for the day in Celsius.
  • temp_low_for_day: Lowest temperature recorded for the day in Celsius.

Example Request:

GET /api/weather

Example Response:

{
    "weather": "clear sky",
    "pressure_in_bar": 1012,
    "temperature_celsius": 24,
    "temperature_fahrenheit": 75.2,
    "icon": "01d",
    "wind": 3.1,
    "sunset": 1624992789,
    "sunrise": 1624946819,
    "temp_high_for_day": 25,
    "temp_low_for_day": 15
}

Note: Ensure your server is running and correctly configured to handle the requests. The response might vary based on the server's location and current weather conditions.
